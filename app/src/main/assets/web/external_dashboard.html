쿠폰 발송

🎫 쿠폰 발송 현황

쿠폰번호 이름 전화번호 이메일주소 발송여부 발송 액션

📋 쿠폰 발송 방법

  1. 발송할 거래처를 선택하세요
  2. 해당 거래처의 쿠폰 목록이 자동으로 로드됩니다
  3. 개별 발송: 각 쿠폰별로 이메일/SMS/카카오톡 버튼 클릭
  4. 일괄 발송: 상단의 전체 발송 버튼 클릭
  5. 발송 상태는 실시간으로 업데이트됩니다
✅ 상태: 쿠폰 발송 기능이 활성화되었습니다. 발송 시 데이터베이스에 기록됩니다.

데이터 백업/복구

📦 데이터 백업

현재 데이터베이스의 모든 데이터를 JSON 파일로 백업합니다.
백업 데이터에는 거래처, 직원, 쿠폰, 거래내역이 포함됩니다.

⚠️ 데이터 복구

주의: 데이터 복구 시 현재 데이터베이스의 모든 데이터가 삭제되고 백업 파일의 데이터로 교체됩니다.
복구하기 전에 반드시 현재 데이터를 백업하세요!

💡 백업/복구 가이드

  • 백업 시점: 정기적으로 백업하여 데이터 손실을 방지하세요.
  • 백업 파일 보관: 백업 파일은 안전한 위치에 보관하세요.
  • 복구 전 확인: 복구할 백업 파일이 올바른 파일인지 확인하세요.
  • 복구 후 확인: 복구 후 데이터가 정상적으로 복원되었는지 확인하세요.
  • 백업 파일 형식: JSON 형식 (.json) 파일만 복구 가능합니다.

🚨 중요 안내

  • 복구 작업은 되돌릴 수 없습니다.
  • 복구 중에는 절대로 앱을 종료하거나 새로고침하지 마세요.
  • 백업 파일이 손상된 경우 복구가 실패할 수 있습니다.
  • 복구 실패 시 데이터가 손상될 수 있으므로 주의하세요.
