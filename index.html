📊 Excel 데이터 일괄 등록

사용 방법:

1. Excel에서 직원 데이터를 선택하여 복사 (Ctrl+C)

2. 아래 텍스트 박스에 붙여넣기 (Ctrl+V)

3. '데이터 미리보기' 버튼을 클릭하여 확인

4. '일괄 등록' 버튼을 클릭하여 저장

데이터 형식 (열 순서):

이름 | 핸드폰번호* | 이메일 | 부서

또는 핸드폰번호* | 이메일 | 부서

예1: 김철수 010-1234-5678 kim@company.com 개발팀

예2: 010-1234-5678 kim@company.com 개발팀

* 핸드폰번호만 필수입니다 (이름 생략 가능)
